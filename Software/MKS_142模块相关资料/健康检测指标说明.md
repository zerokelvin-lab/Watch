# 健康检测指标说明

本文档整理了多项核心健康检测指标的专业定义、正常范围、异常解读、诱因分析及改善建议，同时标注了各指标的本地 / 云端检测输出支持情况。

## 1. 心率

- **指标定义**：心率是指正常人安静状态下每分钟心跳的次数，也叫安静心率，一般为 60～100 次 / 分，可因年龄、性别或其他生理因素产生个体差异。一般来说，年龄越小，心率越快，老年人心跳比年轻人慢，女性的心率比同龄男性快，这些都是正常的生理现象。

- **计量单位**：次 /min

- **适用人群**：静息状态下 10 岁及以上儿童、成人

- **数值范围与等级说明**：

  表格

  

  

  

  |     数值范围     | 健康等级 |                           症状说明                           |
  | :--------------: | :------: | :----------------------------------------------------------: |
  |  60-100 次 /min  |   正常   |                          无异常症状                          |
  | 小于 60 次 /min  | 心动过缓 | 可能有头晕、乏力、倦怠、精神差的症状，运动员或家族遗传一般无异常 |
  | 大于 100 次 /min | 心动过速 | 若心跳过快无法维持有效血液循环，可能出现心慌、气短、乏力等症状 |

  

- **异常原因**：

  - 心动过缓：常见于运动员或家族遗传，其他原因包括心脏供血不足、局部微循环异常、窦房结 / 房室结起搏功能异常等
  - 心动过速：常见于运动、饮酒、吸烟、情绪激动等生理因素，以及感染、发热、贫血、缺氧等病理因素

  

- **改善建议**：

  - 正常范围：继续保持健康生活
  - 心动过缓：实时监测自身心率变化，若出现头晕、乏力、晕厥症状请及时就诊
  - 心动过速：静息状态下测量，戒烟戒酒，增加慢跑、打太极等有氧运动，少熬夜、避免过度劳累，存在病理因素请及时就医

  

- **输出支持**：本地输出支持、云端输出支持

## 2. 血氧饱和度

- **指标定义**：血氧饱和度是血液中被氧结合的氧合血红蛋白的容量占全部可结合的血红蛋白容量的百分比，即血液中血氧的浓度，是呼吸循环的重要生理参数。

- **计量单位**：百分比

- **适用人群**：成人

- **数值范围与等级说明**：

  表格

  

  

  

  | 数值范围 |    健康等级    |                           症状说明                           |
  | :------: | :------------: | :----------------------------------------------------------: |
  |  95-99%  |      正常      |                          无异常症状                          |
  | 小于 95% | 血氧饱和度较低 | 轻度缺氧无明显症状，低于 90% 可能出现呼吸不畅、四肢乏力、头晕、胸闷等症状 |

  

- **异常原因**：呼吸系统或心血管系统可能存在异常，长时间低氧会对神经系统及身体器官造成损伤

- **改善建议**：增加慢跑、打太极等有氧运动，戒烟戒酒，保持富含维生素和铁的绿叶蔬菜（菜花、菠菜等）的健康饮食，避免贫血造成血氧不足

- **输出支持**：本地输出支持、云端输出支持

## 3. 微循环

- **指标定义**：微循环是血管弹性的一种表现形式，血管弹性是维持血管正常生理功能的重要特性，人体血液循环的脉动性就是血流与弹性血管壁持续作用的结果。若血管弹性下降，会对心血管系统产生多方面影响。

- **计量单位**：百分比

- **适用人群**：成人

- **数值范围与等级说明**：

  表格

  

  

  

  | 数值范围 |   健康等级   |                           症状说明                           |
  | :------: | :----------: | :----------------------------------------------------------: |
  | 大于 79% |     正常     |                          无异常症状                          |
  |  70-79%  | 血管指数较低 | 处于亚健康状态，容易出现疲乏无力、情绪低落、睡眠质量差等症状 |
  | 低于 70% | 血管指数过低 | 处于亚健康状态，容易出现疲乏无力、情绪低落、睡眠质量差等症状 |

  

- **异常原因**：

  - 70-79%：亚健康状态，常见于疲劳、虚弱、脂肪摄入过量、缺乏运动等
  - 低于 70%：可能对心血管系统产生多方面影响，容易诱发局部微循环障碍

  

- **改善建议**：低盐低脂饮食，少熬夜、避免过度劳累，戒烟戒酒，增加慢跑、打太极等有氧运动

- **输出支持**：本地输出支持、云端输出支持

## 4. 收缩压（高压）& 舒张压（低压）

### 4.1 收缩压（高压）

- **指标定义**：心室收缩时，主动脉压急剧升高，在收缩期的中期达到最高值，这时的动脉血压值称为收缩压，也称为 “高压”。
- **计量单位**：mmHg
- **适用人群**：成人

### 4.2 舒张压（低压）

- **指标定义**：心室舒张时，主动脉压下降，在心舒末期动脉血压的最低值称为舒张压，也称为 “低压”。
- **计量单位**：mmHg
- **适用人群**：成人

### 4.3 数值范围与等级说明

表格







| 数值范围（收缩压 / 舒张压） | 健康等级 |                           症状说明                           |
| :-------------------------: | :------: | :----------------------------------------------------------: |
|       小于 90/60 mmHg       | 血压较低 | 生理性低血压无明显症状；病理性低血压易出现头晕乏力、眼黑肢软，严重时昏厥 |
|      90-140/60-90 mmHg      |   正常   |                          无异常症状                          |
|      大于 140/90 mmHg       | 血压较高 | 轻度无明显症状，持续升高可能出现头晕、头痛、颈项板紧、疲劳、心悸等症状 |

### 4.4 异常原因

- 低血压：体循环动脉压力低于正常状态，分为生理性和病理性，按起病形式分为急性和慢性
- 高血压：精神紧张、高钠盐低钾饮食、大量饮酒等

### 4.5 改善建议

- 低血压：少熬夜、避免过度劳累，注意补充营养，增加慢跑、打太极等有氧运动
- 正常范围：继续保持健康生活
- 高血压：调节饮食，补钾排钠，增加粗粮摄入；定期测量血压，避免运动后立即测量

### 4.6 输出支持

本地输出支持、云端输出支持

## 5. 心率变异性 HRV

- **指标定义**：心率变异性 (HRV) 是指逐次心跳周期差异的变化情况，含有神经体液因素对心血管系统调节的信息，可判断其对心血管等疾病的病情及预防，是预测心脏性猝死和心律失常性事件的有价值指标。

- **计量单位**：ms

- **适用人群**：成人

- **数值范围与等级说明**：

  表格

  

  

  

  |  数值范围   |    健康等级    |                           症状说明                           |
  | :---------: | :------------: | :----------------------------------------------------------: |
  |  22-120 ms  |      正常      |                          无异常症状                          |
  | 小于 22 ms  | 心率变异性偏低 | 交感神经受刺激活跃，可能导致心跳加速、血压升高、胃肠蠕动减慢等症状 |
  | 大于 120 ms | 心率变异性偏高 |                       可能存在心律失常                       |

  

- **异常原因**：

  - 偏低：心脏自主神经调节功能较差，长期偏低会增加心血管疾病风险
  - 偏高：心律失常或测量过程操作不当

  

- **改善建议**：使用正确的测量方法，少熬夜、避免过度劳累，避免精神压力过大，增加慢跑、打太极等有氧运动

- **输出支持**：本地输出支持、云端输出支持

## 6. 心律失常

- **指标定义**：心律失常是由于窦房结激动异常或激动产生于窦房结以外，激动的传导缓慢、阻滞或经异常通道传导，即心脏活动的起源和（或）传导障碍导致心脏搏动的频率和（或）节律异常。

- **计量单位**：有 / 无

- **适用人群**：成人

- **数值范围与等级说明**：

  表格

  

  

  

  | 检测结果 |     健康等级     |                           症状说明                           |
  | :------: | :--------------: | :----------------------------------------------------------: |
  |    无    |       正常       |                          无异常症状                          |
  |    有    | 可能存在心律失常 | 轻度无明显症状，较严重可能引起心悸、胸闷、头晕、低血压、出汗等症状 |

  

- **异常原因**：心脏活动的起源或传导障碍，导致心脏搏动的频率或节律发生异常

- **改善建议**：若多次测量仍有心律失常现象，请到正规医院检查；同时保持作息规律，避免剧烈运动

- **输出支持**：本地输出不支持、云端输出支持

## 7. 呼吸频率

- **指标定义**：呼吸频率为形容每分钟呼吸次数的医学术语，胸部的一次起伏就是一次呼吸，即一次吸气一次呼气。人体通过呼吸吸进氧气、呼出二氧化碳，维持正常生理功能。

- **计量单位**：次 / 分钟

- **适用人群**：成人

- **数值范围与等级说明**：

  表格

  

  

  

  |     数值范围      | 健康等级 |           症状说明            |
  | :---------------: | :------: | :---------------------------: |
  |  12-20 次 / 分钟  |   正常   |          无异常症状           |
  | 大于 20 次 / 分钟 | 呼吸过速 | 成人呼吸频率超过 20 次 / 分钟 |
  | 小于 12 次 / 分钟 | 呼吸过缓 | 成人呼吸频率低于 12 次 / 分钟 |

  

- **异常原因**：

  - 呼吸过速：剧烈运动、情绪激动、过度紧张等生理因素，以及发热、疼痛、呼吸系统疾病等病理因素
  - 呼吸过缓：心肺功能较强、家族遗传等生理因素，以及药物作用、呼吸系统疾病等病理因素

  

- **改善建议**：

  - 正常范围：继续保持健康生活
  - 呼吸过速：静息状态下测量，开放气道、改善通气和供氧，存在病理因素请及时就医
  - 呼吸过缓：心肺功能较强者继续保持，去除相关诱因，存在病理因素请及时就医

  

- **输出支持**：本地输出不支持、云端输出支持

## 8. 疲劳状态

- **指标定义**：疲劳又称疲乏，是主观上一种疲乏无力的不适感觉，客观上可能在同等条件下，失去完成原来所从事的正常活动或工作的能力。

- **适用人群**：成人

- **数值范围与等级说明**：

  表格

  

  

  

  | 数值范围 | 健康等级 |                       症状说明                       |
  | :------: | :------: | :--------------------------------------------------: |
  | 大于 25  |   正常   |                      无异常症状                      |
  |  15-25   | 轻度疲劳 | 神疲乏力、注意力不集中、专注力差，或精神焦虑、紧张等 |
  | 小于 25  | 过度疲劳 |  周身乏力、记忆力减退、情绪波动较大、睡眠质量较差等  |

  

- **异常原因**：工作 / 学习任务重、运动过量、睡眠不足等生理性疲劳，或心理焦虑、悲观压抑等心理性疲劳

- **改善建议**：保证睡眠充足，补充营养，注意劳逸结合，调节心情、疏解压力，增加跑步、打太极等有氧运动

- **输出支持**：本地输出支持、云端输出支持

## 9. 心率散点图

- **指标定义**：心率散点图是采用非线性分析和计算机技术，将长时间记录的心率数据转换成高密度、集中表达的图形，是海量心率数据分析（尤其是动态心率分析）的实用新技术新方法。
- **适用人群**：成人
- **临床应用**：临床医生诊断时，仅需浏览图形，观察分析图形分布数目、形状及 B 线斜率等，即可准确区分心房扑动、室上性早搏、持续和发作性心房颤动、持续心房颤动伴室性早搏等心脏疾病。
- **输出支持**：本地输出不支持、云端输出支持
